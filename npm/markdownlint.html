web-fe-markdownlint

TIP

MD文档 规范

支持配套的 markdownlint 可共享配置。

安装

需要先行安装 markdownlint：

npm install web-fe-markdownlint markdownlint --save-dev

使用

在 .markdownlint.json 中继承本包:

{
  "extends": "web-fe-markdownlint"
}
